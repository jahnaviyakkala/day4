Objective:

The objective of this task is to design an interactive sales dashboard using Power BI that helps business stakeholders understand sales performance, profitability, and trends to support data-driven decision-making.

Dataset:

Dataset Name: Sample Superstore – Sales & Financial Dataset
Source: Kaggle

The dataset contains historical sales transaction data with information related to order dates, sales amount, profit, product categories, and regional performance.

Tool Used:

Microsoft Power BI Desktop

Key Performance Indicators (KPIs):

The following KPIs were used to summarize business performance:

Total Sales – Overall revenue generated

Total Profit – Total profitability

Profit Margin (%) – Ratio of profit to total sales

Dashboard Features:

KPI Cards displaying Total Sales, Total Profit, and Profit Margin

Line Chart for Monthly Sales Trend analysis

Bar Chart showing Sales by Product Category

Column Chart showing Profit by Region

Interactive slicers for filtering data by Region and Category

The dashboard updates dynamically based on user selections.

Key Business Insights:

Overall sales show a declining trend over time

The South region generates higher profit in filtered views

Technology category contributes the highest sales

The overall profit margin is approximately 13%, indicating moderate profitability

Outcome & Learning:

Through this task, I learned how to:

Design an interactive dashboard using Power BI

Create and use measures using DAX

Implement slicers for dynamic filtering

Present business insights visually for stakeholders

This dashboard provides a clear and interactive view of sales performance and profitability, enabling stakeholders to identify trends, high-performing categories, and regional insights effectively.
